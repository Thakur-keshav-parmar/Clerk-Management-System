Coaching Institute Management System — Installation Guide

📋 Requirements before you start

  • Windows PC with XAMPP installed  →  xampp-windows-x64-*.exe
  • XAMPP version 7.4 or higher (PHP 8+ recommended)
  • At least 200 MB free disk space
  • The zip file: clerk-management-system.zip

🔧 Installation Steps

1

Install XAMPP (skip if already installed)

Download XAMPP from apachefriends.org and install it. The default install location is C:\xampp\. Keep all defaults during installation.

2

Start Apache and MySQL

Open XAMPP Control Panel (search for it in Start menu or open C:\xampp\xampp-control.exe).

  • Click Start next to Apache
  • Click Start next to MySQL
  • Both rows should turn green
3

Extract the ZIP file

Right-click clerk-management-system.zip → Extract All…

  • Extract to: C:\xampp\htdocs\
  • You should now have the folder: C:\xampp\htdocs\clerk-management-system\
  • Make sure index.html is directly inside that folder (not nested deeper)
⚠ If a folder called clerk-management-system already exists, delete or rename it first before extracting.
4

Run the One-Click Installer

Open any browser (Chrome / Edge) and go to:

http://localhost/clerk-management-system/install.php

This will automatically:

  • Create the coaching_db database
  • Import all tables and data
  • Apply all required migrations
  • Set up default login accounts
✓ When you see "Installation Complete!" the setup is done.
5

Delete the installer (important!)

After installation is complete, delete the file:

C:\xampp\htdocs\clerk-management-system\install.php
⚠ Leaving install.php accessible is a security risk — anyone who visits that URL can reset/overwrite the database.
6

Open the Application

Go to:

http://localhost/clerk-management-system/

Login with these default credentials:

RoleUsernamePassword
👑 Adminadminadmin123
📚 Teacherteacherteacher123
⚠ Change the admin password after first login via Settings → User Management.

🛠 Troubleshooting

!

Apache or MySQL won't start

  • Port 80 may be in use — try changing Apache to port 8080 in XAMPP → Config → httpd.conf
  • Port 3306 may be in use — check if another MySQL is running (Wamp, local MySQL service)
  • Run XAMPP Control Panel as Administrator
!

install.php shows a database error

  • Make sure MySQL is running (green in XAMPP Control Panel)
  • Make sure the database/ folder inside the zip contains coaching_db.sql
  • Try opening: http://localhost/phpmyadmin — if it loads, MySQL is fine
!

Page shows 404 / "Not Found"

  • Make sure the folder name is exactly clerk-management-system (no spaces, lowercase)
  • Confirm Apache is running (green in XAMPP)
  • Check that index.html exists at C:\xampp\htdocs\clerk-management-system\index.html
